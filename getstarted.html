Instalación desde código fuente

Instalación para Windows

Para la Implementación del sistema desde sus paquetes fuentes necesitamos descargar otras herramientas para un correcto funcionamiento del sistema.

1.Instalar MongoDB

MongoDB es un sistema de base de datos NoSQL, orientado a documentos y de código abierto. En lugar de guardar los datos en tablas, tal y como se hace en las bases de datos relacionales

Para su instalación debe de ir a la pagina oficial de MongoDB y selecciona el instalador correspondiente a su sistema operativo.

2.Instalar NodeJS

Necesitamos instalar Node.js ya que luego instalaremos loopback mediante npm

Para su instalación nos dirigimos al siguiente enlace https://nodejs.org/en/download y seleccionar el instalador o binario respectivo.

3.Instalar Loopback

Para la instalación de loopback, primero verificamos que tengamos instalado Node.js y npm.

Una vez hayamos verificado la correcta instalación de npm ejecutamos

npm install -g @loopback/cli

y Ya tendriamos todas las herramientas instaladas

Una vez instaladas todas las herramientas procedemos a descargar los paquetes fuentes

Configuración de entorno del sistema ResBar

Procedemos a descargar los archivos de la base de datos clickeamos en Download

Esto le descargara un archivo .zip, que debera descomprimir. Opcionalmente tambien puede descargarlo mediante git y hacer un git clone al enlace de github https://github.com/ResBar-2020/Resbar-backend-v1.0.git.

Cualquier método nos dejará dos directorios, uno con archivos referentes a la base de datos y el otro del API.

Luego debemos de iniciar el servicio de mongodb, ejecutando en la terminal como administrador el comando:

mongod

Una vez iniciado el servicio de mongo procedemos a hacer un restore de la base de datos, En la terminal ejecutamos el siguiente comando:

mongorestore --db=resbar --dir=/home/kevin/Descargas/Resbar-backend-v1.0/resbar-backup/resbar/

Debera cambiar algunos parametros:

  • --db indica el nombre de la base de datos
  • --dir indica la ruta absoluta del directorio donde se han descargado los archivos de la base de datos.

Luego nos dirigimos al directorio donde descargamos el api, en nuestro caso seria:

cd Descargas/Resbar-backend-v1.0/resbar-apirest2020/

Una vez estando acá, ejecutamos el comando npm install, para descargar todas las dependencias del proyecto

Una vez descagadas todas las dependencias,ejecutamos un npm start para levantar el servidor del API.

Ahora descargamos el codigo fuente desde el siguiente enlace https://github.com/ResBar-2020/ResBar.git

una vez descargado ya puede ejecutar el sistema ResBar.
